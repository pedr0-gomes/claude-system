**The Missing Semester of Your CS Education** (MIT CSAIL) — https://missing.csail.mit.edu/

A cadeira sobre as ferramentas de CS que o currículo formal não ensina. Aulas (versão 2026): shell, command-line environment, dev environment & tools, debugging & profiling, git/version control, packaging & shipping, agentic coding, beyond the code, code quality.

**Como Pedro quer usar:** NÃO de modo convencional (assistir aulas + atividades). É **mapa de temas** pra entrada *just-in-time* do bloco Aprender — quando uma task real (Linux, git/github, vercel, supabase, encanamento de dev) bater num desses essenciais, puxar o assunto a partir dela. A task é o gatilho; o Missing Semester nomeia o que vale a parada. Motivo de fundo: construir autonomia técnica real (perfil) — saber fazer por si o que hoje o Claude faz por ele.
